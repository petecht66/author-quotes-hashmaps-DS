# author-quotes-hashmaps-DS
Date Submitted: November 21, 2024
Last Updated on GitHub: April 7, 2026

This repository contains my complete work for the hashmaps project of CISC220: Data Structures at the University of Delaware College of Engineering Department of Computer and Information Sciences. I completed this course in the Fall of 2024, with Professor Leighanne Hsu being the course instructor. The purpose of this course was to introduce students to topics such as data type abstraction, recursion, arrays, stacks, linked lists, dynamic storage management, trees, graphs, tables, sorting, searching, and more.

This specific project dealt with hashmaps, a key concept for the course. The objective of this project was for students to write their own hashmaps to generate text from two authors: Dr. Seuss and Charles Dickens. Each word read in was a key, and the values associated with each key was an array of strings, or the set of words that follow a word. In addition to creating the hashmaps for these authors, students were also required to write 2 hash functions and 2 collision functions to compare efficiency.

Four folders were created for this repository. The 'header-source-files' folder contains the three C++ header source files given to students from Professor Hsu. Next, the 'source-files' folder contains the four C++ source files that I had to complete for this project. The 'Charles-Dickens-output' contains the four output files for the Charles Dickens quotes. The 'Dr-Seuss-output' folder contains the four output files for the Dr. Seuss quotes.
